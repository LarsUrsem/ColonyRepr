# Species specific smoothing parameters

A data frame containing species-mean smoothing parameters, calculated
from smoothing parameters determined using LSCV per winter track

## Usage

``` r
mean_smooths
```

## Format

...

- species:

  smoothing parameters (h_par) are provided for 15 seabird species

- mean_h:

  mean values from track-specific smoothing parameters determined using
  LSCV
