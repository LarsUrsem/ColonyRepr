# Example data as would be returned from \`KDE_saturation_curve_fun()\`

A data frame containing information on combined contours surface areas
and the parameters of the Michaelis-Menten curve fitted to this data.

## Usage

``` r
saturation_curve
```

## Format

...

- species:
- colony:
- month:
- n_inds:
- it:
- perc:
- area:
- h_par:
- A_mean:
- B_mean:
- Repr:
