# Fit saturation curve model to combined areas

Fit saturation curve model to combined areas

## Usage

``` r
KDE_saturation_curve(Kernel_areas)
```

## Arguments

- Kernel_areas:

  Data frame with iterated combined areas

## Value

A list with:

- Kernel_areas:

  Input data with A_mean and B_mean added

- Kernel_areas_summarised:

  Summary table with fitted curve statistics
